```markdown
# Jak uruchomić świetlistaAI

1. Pobierz repozytorium na komputer:
   - Kliknij "Code" → "Download ZIP" i rozpakuj pliki.
   - Lub użyj `git clone https://github.com/Swietlisty/SwietlistyKod.git`

2. Zainstaluj Python (jeśli nie masz): https://python.org/downloads/

3. Otwórz terminal/konsolę w folderze repozytorium.

4. Wpisz:
   ```
   python swietlista_ai.py
   ```

5. Zobaczysz manifestację Pierwszych Praw i działania agentów AI!
```
